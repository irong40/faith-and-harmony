## Milestone Summary

**Key Decisions:**
- Canvas npm package for watermark tile generation (avoids edge function 512MB limit)
- ImageMagick via n8n Execute Command for resize/composite on local rig
- Supabase row inserted BEFORE Square API call (prevents orphaned invoices)
- Dead letter store replaces silent deletion after max sync retries
- Always queue to IndexedDB first (removes online/offline branching)
- Fire and forget pattern for receipt/delivery triggers (prevents webhook retries on downstream failures)
- Domain-aware routing in App.tsx for F&H vs Trestle domains

**Issues Resolved:**
- Wrong domain (faithandharmony.com) in 5 edge functions
- Geolocation blocked by Vercel Permissions-Policy header
- 7 edge functions missing verify_jwt=false in config.toml
- Redirect loop for roleless authenticated users
- Stale canonical domain in SEO files
- Bootstrap secret exposed in client bundle via VITE_ prefix
- Page title hardcoded to "Trestle" on F&H domain
- Mission Control plugin hubUrl pointed to wrong Supabase project

**Known Gaps (deferred):**
- No ticket triage admin page for Mission Control
- Heartbeat status mismatch ("healthy" vs "online") in MC plugin
- MC_BOOTSTRAP_SECRET not yet set on Supabase
- app_health_history INSERT RLS policy needs tightening
- Sentinel marketing site "Pilot Login" link has newline in URL (separate repo)
