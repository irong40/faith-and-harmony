async function handleGetPackagePricing(args: { service_type: string }): Promise&lt;string&gt;

The function does:

2. If no service_type provided, return "I need to know which service you are asking about. Could you tell me the package name?"

  450 becomes "four hundred fifty dollars"
  750 becomes "seven hundred fifty dollars"
  850 becomes "eight hundred fifty dollars"
  1200 becomes "twelve hundred dollars"

Wire the handler into the for loop in the serve() callback, adding an else-if branch after the existing lookup_customer check:
