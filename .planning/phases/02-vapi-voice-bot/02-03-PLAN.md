  Navigate to Vapi Dashboard. Go to Provider Keys (or Integrations). Find ElevenLabs. Paste the API key. Save. Wait a moment for voice library to sync.

Section "Step 2: Create the Assistant"
  Go to Assistants. Click Create. Set the name to "Paula - Sentinel Aerial Intake". Under Model, select OpenAI and gpt-4o-mini. Set temperature to 0.7 and max tokens to 250. Set End Call Message to "Thank you for calling Sentinel. We will follow up with you shortly. Have a great day."

Section "Step 3: Configure the Voice"
  Under Voice, select ElevenLabs as the provider. The voice library should now show synced voices. Browse the catalog and select a professional female American voice. Recommended categories to search: "young professional" or "American female." Note the voiceId for reference. Set model to "eleven_flash_v2_5". Set speed to 1.0, stability to 0.5, similarity to 0.75. Test the voice with the preview button before saving.

Section "Step 4: Add Tool Definitions"
  Under Tools, add the two tools from tools.json. Section "Step 5: Configure Timeouts and Settings"
  Set Silence Timeout to 30 seconds. Set Max Duration to 600 seconds (10 minutes). Enable Background Denoising. Save the assistant.

Section "Step 6: Provision 757 Phone Number"
  Go to Phone Numbers. Click Create Phone Number. Select the Free Vapi Number tab. Enter area code 757. Click Create. Wait approximately 5 minutes for activation. After activation, select the new number. Under Inbound Call Handler, set to Assistant and select "Paula - Sentinel Aerial Intake" from the dropdown. Save.

Continue through the qualification flow. Ask about pricing for Listing Pro. Paula should call the get_package_pricing tool and respond with "four hundred fifty dollars" and the deliverables. End the call.
