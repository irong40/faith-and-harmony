Replace vercel.json with the following content. The rewrites array is preserved exactly. A new headers array is added that applies to all routes (source: "/(.*)" matches every path).

```json
{
  "rewrites": [
    { "source": "/(.*)", "destination": "/index.html" }
  ],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "X-Frame-Options",
          "value": "DENY"
        },
        {
          "key": "X-Content-Type-Options",
          "value": "nosniff"
        },
        {
          "key": "Referrer-Policy",
          "value": "strict-origin-when-cross-origin"
        },
        {
          "key": "Strict-Transport-Security",
          "value": "max-age=63072000; includeSubDomains; preload"
        },
        {
          "key": "Permissions-Policy",
          "value": "camera=(), microphone=(), geolocation=()"
        }
      ]
    }
  ]
}
```

Header rationale:
- X-Frame-Options DENY: prevents clickjacking. The Sentinel landing page has no legitimate iframe use case.
- X-Content-Type-Options nosniff: prevents MIME-type sniffing attacks on script and style responses.
- Referrer-Policy strict-origin-when-cross-origin: sends origin only on cross-origin requests, protecting query parameters from leaking to third parties.
- Strict-Transport-Security max-age=63072000 includeSubDomains preload: 2-year HSTS with subdomain coverage and preload eligibility. Faith & Harmony LLC operates exclusively over HTTPS.
- Permissions-Policy: disables camera, microphone, and geolocation browser APIs. The landing page does not use any of these.

Do NOT add a Content-Security-Policy header. The landing page loads Google Fonts from fonts.googleapis.com and fonts.gstatic.com, embeds inline styles via Tailwind/React, and uses the gptengineer.js script from cdn.gpteng.co. A permissive CSP would be less effective than none, and a restrictive one requires significant testing before deployment. CSP is deferred to v2.

Do NOT add X-XSS-Protection. This header is deprecated in modern browsers and can cause issues on older IE.
