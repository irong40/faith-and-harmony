  <action>
    Create src/hooks/useWeatherForecast.ts with two hooks:

    1. useWeatherForecast():
       - queryKey: ['weather-forecast']
       - queryFn: select * from weather_forecast_cache where forecast_hour >= now and forecast_hour <= now + 48 hours, order by forecast_hour ascending
       - staleTime: 30 * 60 * 1000 (30 minutes, since the daily cron means cache is usually good)
       - Export a ForecastRow type with all columns from weather_forecast_cache (id, forecast_hour, wind_speed_ms, wind_gust_ms, visibility_sm, cloud_ceiling_ft, precipitation_probability, temperature_c, sky_cover_pct, determination, determination_reasons)

    2. useWeatherHeldJobs():
       - queryKey: ['weather-held-jobs']
       - queryFn: select id, job_number, scheduled_date, scheduled_time, status, weather_hold, weather_hold_reasons from drone_jobs where weather_hold = true and status not in ('delivered', 'cancelled'), order by scheduled_date ascending
       - staleTime: 5 * 60 * 1000 (5 minutes)
       - Return the data typed appropriately

    Create src/pages/admin/components/WeatherForecastGrid.tsx:

    A table component that receives ForecastRow[] and renders a scrollable grid:

    - Each row is one forecast hour
    - Columns: Time (formatted with date-fns format as "EEE ha" like "Mon 2p"), Wind (sustained m/s), Gust (m/s), Visibility (SM), Ceiling (ft), Precip (%), Sky Cover (%), Status (GO/CAUTION/NO_GO badge)
    - Use lucide icons in column headers: Wind for wind columns, Eye for visibility, Cloud for ceiling/sky cover, Thermometer (not used, but import pattern reference)
    - Status badge colors: GO = green background white text, CAUTION = amber/yellow background white text, NO_GO = red background white text. Match the DETERMINATION_CONFIG pattern from WeatherBriefingPanel.tsx.
    - Highlight rows with NO_GO determination with a subtle red background tint (bg-red-50)
    - Highlight rows with CAUTION with a subtle amber background tint (bg-amber-50)
    - When a value is null, display a dash character
    - Round wind values to 1 decimal. Round visibility to 1 decimal. Ceiling and precip are integers.
    - Use Shadcn Table component (Table, TableBody, TableCell, TableHead, TableHeader, TableRow) for consistent styling with the rest of the admin
    - Use Shadcn Badge for determination status
