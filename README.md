# Sentinel Aerial Inspections — Trestle

Field operations command center for Sentinel Aerial Inspections drone services.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (auth, database, edge functions)
- PWA with offline support

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deployment

Deployed via Vercel with git integration — pushes to `main` auto-deploy.
